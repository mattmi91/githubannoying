Pricing options, use these cost examples to calculate an estimate for the build you are looking to get

Main computer choices to run everything

The main price point difference in the entire build revolves around two different computer choices, the first choice would be a 35$ single board computer that can run every system and game besides PS2/Gamecube/Wii/PSP the cheaper alternative single-board computer and ones similar around the 50$ price tag simply do not have the required power to run these more demanding game systems, but can still play most of the video game systems on the list, then there is the real gaming pc option that will be able to play every system on the list and anything you throw at it including modern graphical intensive video games that are out today this is gonna be around 400-500$ price tag for the gaming pc there's no other way around it, video game emulation is difficult and the more graphic intensive games you want to play you need better hardware to do it.

Wood to build the cabinet

The two 19 inch builds with the small tabletop and the stand up 19-inch screen the wood cost is gonna be about 100$ for those as for the 32 inch and 55 inch builds the wood cost is gonna range from 100-200$

Arcade Joysticks, buttons, Arcade Guns & controllers

The cost for 2 player joysticks and buttons cost about 30-50$ for a set if you want 4 players you will have to buy 2 sets of this if you already own Xbox 360, ps3, ps4, Xbox one controllers if you want USB hookup with a cord then these will be free since you already have them, if you want wireless you will have to buy a controller adapter for about 20$ for the light gun arcade set to be able to play shooting games that you point the gun at the screen to aim, Wii remotes will be used along with a USB Wii sensor bar so if you don't already own Wii remotes you will have to buy these. The Wii remote setup works for all light gun arcade games and of course Wii games themselves. Guitar hero guitars and rock band drums will also work with the cabinet to play those music games that were on Playstation 2 and Nintendo Wii
