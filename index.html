Cheaper alternaves vs best of the best

Price points vary I sell several cheaper alternatives such as the game console for around 140$ and mini arcade for 250$ these cheaper alternativers will play every system on the list besides PS2/GAMECUBE/PSP/DS you will need a gaming pc to play these systems which can range for about 500$ the cheaper alternatives still have 13,000 games so you get quite a lot, where my gaming pc versions have about 25,000 games with arcade guns and are some of the best fully loaded arcade machines in the entire world.

Wood to build the cabinet

The two 19 inch builds with the small tabletop and the stand up 19-inch screen the wood cost is gonna be about 100$ for those as for the 32 inch and 55 inch builds the wood cost is gonna range from 100-200$, my cheaper alternatives are all 3d printed so no wood cost for those

Arcade Joysticks, buttons, Arcade Guns & controllers

The cost for 2 player joysticks and buttons cost about 30-50$ for a set if you want 4 players you will have to buy 2 sets of this if you already own Xbox 360, ps3, ps4, Xbox one controllers if you want USB hookup with a cord then these will be free since you already have them, if you want wireless you will have to buy a controller adapter for about 20$ for the light gun arcade set to be able to play shooting games that you point the gun at the screen to aim, Wii remotes will be used along with a USB Wii sensor bar so if you don't already own Wii remotes you will have to buy these. The Wii remote setup works for all light gun arcade games and of course Wii games themselves. Guitar hero guitars and rock band drums will also work with the cabinet to play those music games that were on Playstation 2 and Nintendo Wii
